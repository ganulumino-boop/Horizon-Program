## 🚀 ARES-1 Spacecraft

ARES-1 is a vertical integrated spacecraft — the most ambitious crewed vehicle ever designed.

```
┌─────────────────────┐
│   CREW CAPSULE      │  20m — 8 seats — dome shaped
├─────────────────────┤
│   LIVING ROOM       │  30m — habitable section — large windows
├─────────────────────┤
│                     │
│   SERVICE MODULE    │  100m — 4 segments — avionics, power,
│                     │         LH2 tanks, structural base
│                     │
├─────────────────────┤
│   NTP ENGINE BLOCK  │  12m — 3 main engines — Isp ~900s
└─────────────────────┘
        150m TOTAL
```

### Seven-Module Orbital Assembly

| Module | Length | Function |
|--------|--------|----------|
| M1 — Crew Capsule | 20m | Human-rated — crew boards here |
| M2 — Habitat/Living Room | 30m | Pressurised crew living space |
| M3 — Service Seg A | 28m | Life support, avionics, science |
| M4 — Service Seg B | 28m | Power systems, water, comms |
| M5 — Service Seg C | 28m | LH2 propellant tanks |
| M6 — Service Seg D | 16m | Structural interface, feed lines |
| M7 — NTP Engine Block | 12m | Nuclear reactor, 3 main engines |

## 🗺️ Mission Flight Plan — 10 Phases

| Phase | Duration | Key Event | Delta-V |
|-------|----------|-----------|---------|
| 1 — Orbital Assembly | Weeks-months | ARES-1 built from 7 modules | — |
| 2 — Earth Refueling | Days | Propellant loaded — crew boards | — |
| 3 — Trans-Mars Injection | 20-40 min | Point of no return | ~3,600 m/s |
| 4 — Outbound Transit | ~200-260 days | Deep space cruise | ~200 m/s |
| 5 — LMO Depot Rendezvous | ~3-5 days | Earth return propellant loaded | ~50 m/s |
| 6 — Mars Orbital Insertion | 20-40 min | First humans to orbit Mars | ~900 m/s |
| 7 — Mars Orbital Operations | TBD | Full science program | ~200 m/s |
| 8 — Trans-Earth Injection | 20-40 min | Departure from Mars | ~1,100 m/s |
| 9 — Return Transit | ~200-260 days | Return journey | ~100 m/s |
| 10 — Reentry & Splashdown | ~30 min | Crew recovery | Aerobraking |
| **TOTAL** | **~500-580 days** | **All 8 crew home alive** | **~5,900-6,200 m/s** |

## ⚠️ The Failure Clause

> If Oblivion I — the crewed mission — fails for any reason under any circumstances, the Horizon Program receives **zero further funding** and is immediately and permanently dissolved. Eight sovereign nations absorb one trillion dollars in unrecoverable debt.

**This clause does NOT apply to Oblivion I-T.**
